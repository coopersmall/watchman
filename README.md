# Watchman

Blockchain and investments aggregator

# Setup

## Install NVM

### Homebrew

```sh
brew install nvm
```

## Install Nod

### Homebrew

```sh
brew install node
```

## Install Yarn

### Homebrew

```sh
brew install yarn
```

# Helpful Commands

## Install Dependencies

```sh
make deps
```

## Start App

```sh
make start
```

## Check Lint

```sh
make lint
```

## Fix Lint

```sh
make relint
```

## Apply Formatting

```sh
make format
```

# Testing

## Unit Test

```sh
make test
```
